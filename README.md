# A-Share Quant Lens

A-Share Quant Lens 是一个面向普通投资者的 A 股量化可视化工具原型。项目目标是把指数走势、行业强弱、因子权重、资金流和策略信号合并到一个低门槛界面里，让非专业用户也能理解量化研究的基本过程。

这个仓库可作为小米 MiMo Token 计划的审核材料：申请者是浙江大学学生，希望使用大模型能力把复杂金融数据转换为可解释的投资研究视图，并以开源方式分享最小可运行界面。

## Demo Scope

- A 股市场总览与核心指数看板
- 动量、估值、质量、流动性等因子筛选
- 行业热力图与样例股票池
- 面向普通投资者的自然语言策略解释入口

当前版本使用模拟数据，便于审查界面与交互。后续计划接入公开行情、财报和研报摘要数据，增加回测、风险归因、组合再平衡和大模型解释模块。

## Local Preview

Open `index.html` directly in a browser.

## Roadmap

1. 接入 A 股指数、行业和个股公开数据源。
2. 增加因子回测和组合收益归因。
3. 使用大模型生成通俗解释、风险提示和研究摘要。
4. 加入教育模式，帮助普通股民理解量化模型的边界。
